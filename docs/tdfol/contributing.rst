Contributing
============

We welcome contributions to TDFOL!

Getting Started
---------------

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests for new functionality
5. Ensure all tests pass
6. Submit a pull request

Development Setup
-----------------

.. code-block:: bash

    git clone https://github.com/ipfs-datasets/ipfs_datasets_py.git
    cd ipfs_datasets_py
    pip install -e ".[test]"
    pytest tests/unit_tests/logic/TDFOL/

Coding Standards
----------------

- Follow PEP 8
- Add type hints
- Write comprehensive docstrings
- Include unit tests

See CONTRIBUTING.md in the repository root for more details.
