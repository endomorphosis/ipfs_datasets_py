Installation
============

Quick Install
-------------

.. code-block:: bash

    pip install ipfs-datasets-py

From Source
-----------

.. code-block:: bash

    git clone https://github.com/ipfs-datasets/ipfs_datasets_py.git
    cd ipfs_datasets_py
    pip install -e .

Dependencies
------------

TDFOL requires Python 3.12+ and has the following dependencies:

- Core dependencies are installed automatically with pip
- Optional dependencies for visualization and ZKP features

See :ref:`tutorial-getting-started` for more details.
