Changelog
=========

Version 1.0.0 (2024)
--------------------

Initial release with:

- Complete TDFOL formula representation
- Multiple proof strategies (Resolution, Tableaux, Natural Deduction)
- Modal logic support (K, T, S4, S5)
- Temporal logic (LTL, CTL)
- Deontic logic (SDL)
- Zero-knowledge proof integration
- Performance optimization with caching
- Comprehensive visualization tools
- 100+ unit tests with 98%+ coverage

See the project repository for detailed commit history.
