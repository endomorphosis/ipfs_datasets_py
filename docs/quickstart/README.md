# Quick Start Guides

Fast-track guides to get started with specific features quickly.

## Available Quick Starts

Quick start guides in this directory provide rapid onboarding for specific features or components.

## Purpose

Quick starts are designed to:
1. **Get You Running Fast** - 5-15 minute setup
2. **Focus on One Thing** - Single feature or task
3. **Minimal Prerequisites** - Quick installation
4. **Working Examples** - Copy-paste ready code

## Using Quick Starts

1. Choose the quick start for your need
2. Follow the setup steps
3. Run the example code
4. Explore further with full documentation
